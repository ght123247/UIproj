# 主题配色方案说明

## 当前配色方案

**方案1: 赛博朋克风格** (当前使用)
- **主色1 (orange-500)**: `#ff00ff` - 霓虹紫
- **主色2 (blue-500)**: `#00b8ff` - 霓虹蓝
- **特点**: 强烈的未来科技感，适合科幻风格界面
- **适用场景**: 高科技监控、未来感仪表盘

## 其他配色方案

### 方案2: 工业机械风格
- **主色1 (orange-500)**: `#ff6b00` - 警告橙
- **主色2 (blue-500)**: `#3498db` - 金属蓝
- **特点**: 经典工业监控配色，专业稳重
- **适用场景**: 工业监控、机械设备控制

### 方案3: 未来科技风格
- **主色1 (orange-500)**: `#00ff00` - 能量绿
- **主色2 (blue-500)**: `#00d4ff` - 电光蓝
- **特点**: 高对比度，充满能量感
- **适用场景**: 能源监控、高科技设备

### 方案4: 暗黑工业风格
- **主色1 (orange-500)**: `#ffb800` - 警示黄
- **主色2 (blue-500)**: `#566573` - 钢铁灰蓝
- **特点**: 低调沉稳，重工业感
- **适用场景**: 重工业监控、暗黑风格界面

### 方案5: 原始配色
- **主色1 (orange-500)**: `#e04671` - 粉红
- **主色2 (blue-500)**: `#46e0c7` - 青绿
- **特点**: 温和优雅，之前使用的配色
- **适用场景**: 通用监控界面、温和风格

## 如何切换配色方案

1. 打开 `tailwind.config.js` 文件
2. 找到 `// 方案1: 赛博朋克风格 (当前使用)` 部分
3. 将当前方案注释掉，取消注释你想要使用的方案
4. 保存文件，前端会自动重新编译

### 示例：切换到工业机械风格

```javascript
// 注释掉方案1
// orange: {

//   ...
// },

// 取消注释方案2
orange: {

```

## 配色说明

- **orange-500**: 用于 `isOrange={true}` 的组件（如 RPM、POWER 等）
- **blue-500**: 用于 `isOrange={false}` 的组件（如 TORQUE、RMS 等）
- **400**: 用于边框和浅色显示
- **600**: 用于 hover 状态
- **900**: 用于深色文字（进度条内的百分比文字）

## 自定义配色

如果需要自定义配色，只需修改对应的颜色值即可。建议：
- 保持足够的对比度以确保可读性
- 主色1和主色2应该有明显的区分度
- 考虑在深色背景下的显示效果
