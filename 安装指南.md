# 开发环境安装指南

## 第一步：安装 Node.js 和 npm

1. **访问 Node.js 官网**: https://nodejs.org/
2. **下载 LTS 版本**（推荐，长期支持版本）
3. **运行安装程序**，安装时：
   - 保持默认选项
   - 确保勾选 "Add to PATH"（通常默认已勾选）
4. **安装完成后**，重新打开 PowerShell 或命令提示符

**验证安装**:
```bash
node --version
npm --version
```

## 第二步：安装 Python

1. **访问 Python 官网**: https://www.python.org/downloads/
2. **下载 Python 3.8 或更高版本**
3. **运行安装程序**，安装时：
   - ⚠️ **重要**: 勾选 "Add Python to PATH"
   - 选择 "Install Now" 或自定义安装
4. **安装完成后**，重新打开 PowerShell 或命令提示符

**验证安装**:
```bash
python --version
python -m pip --version
```

## 第三步：检查环境

运行环境检查脚本：
```powershell
.\check-env-simple.ps1
```

或者使用批处理文件：
```cmd
check-env-simple.bat
```

## 第四步：安装项目依赖

确认 Node.js 和 Python 都已安装后，运行：
```powershell
.\install-deps.ps1
```

这个脚本会自动：
- 安装前端依赖（npm install）
- 安装后端依赖（pip install）
- 创建环境配置文件

## 第五步：启动开发服务器

### 启动后端（在一个终端窗口）:
```powershell
.\dev-backend.ps1
```
后端将在 http://localhost:8000 启动
API 文档: http://localhost:8000/api/docs

### 启动前端（在另一个终端窗口）:
```powershell
.\dev-frontend.ps1
```
前端将在 http://localhost:5173 启动

## 常见问题

### 1. 安装 Node.js 后仍然提示未找到

**解决方案**:
- 关闭所有 PowerShell/命令提示符窗口
- 重新打开新的窗口
- 如果还是不行，检查系统环境变量 PATH 是否包含 Node.js 安装路径

### 2. 安装 Python 后仍然提示未找到

**解决方案**:
- 确保安装时勾选了 "Add Python to PATH"
- 如果没有勾选，需要手动添加到系统环境变量
- 或者重新安装 Python，这次记得勾选

### 3. npm install 很慢或失败

**解决方案**:
- 如果已配置代理，确保代理正常工作
- 可以尝试使用国内镜像：
  ```bash
  npm config set registry https://registry.npmmirror.com
  ```

### 4. pip install 很慢或失败

**解决方案**:
- 可以尝试使用国内镜像：
  ```bash
  python -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
  ```
